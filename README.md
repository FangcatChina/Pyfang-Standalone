# Pyfang Standalone

#### 介绍
让Pyfang语言文件独立运行！

#### 软件架构

使用Pyfang源代码，保留了pyfangrun函数以及valuecheck函数并生成exe文件进行独立运行。


#### 安装教程

直接使用

#### 使用说明

1.  将主程序放入单独文件夹中
2.  在文件夹下新建file.pfr和while.con文件
3.  在file.pfr文件中写入Pyfang代码，在while.con中设置是否循环执行程序，是为True，否为False。
4.  双击主程序即可独立运行Pyfang程序。
